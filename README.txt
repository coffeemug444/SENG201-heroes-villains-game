HOW TO RUN THE GAME
The game is contained within the jar file "<insertname here>.jar".
There are two methods to run this:
1. Double click the .jar file
2. Open your systems terminal, go to the folder where the .jar is located and use the command:
	"java -jar <insertname here>.jar


HOW TO BUILD THE SOURCE CODE
The game is written in Java. The games files are located in the folder <name>.
This can be loaded into Eclipse by going file>Import>general>File System then choosing the folder to import as a project.
Once the project has imported, it can be run by opening and running the GameEnvironment class.
